# Shell, basics
This project is an introduction to the Shell and basic Linux commands.
* **0-current_working_directory**: Prints the absolute path name of the current working directory.
* **1-listit**: Display the contents list of your current directory.
* **2-bring_me_home**: Add script that change directory to home.
*  **3-listfiles**: Add script to list directory contents in long format.
*  **4-listmorefiles**: Add script to list hidden files in the directory.
*  **5-listfilesdigitonly**: Add script to list all files with numerical IDs in long format.
*  **6-firstdirectory**: Add script to create a directory in /tmp.
*  **7-movethatfile**: Add script to move the file betty to my_first_directory.
*  **8-firstdelete**: Add script to delete the file betty.
*  **9-firstdirdeletion**: Add script to delete the directory my_first_directory.
*  **10-back**: Add a script that changes the working directory to the previous one.
*  **11-lists**: Add script to list all files in current, parent and /boot/directories.
*  **12-file_type**: Add script to print the type of the file iamafile.
*  **13-symbolic_link**: chore: create symbolic link __ls__  pointing to /bin/ls
*  **14-copy_html**: feat: add html sync script to parent directory.
*  **15-lets_move**: feat: move files starting with uppercase to /tmp/u
*  **16-clean-emacs**: feat: delete files ending with ~.
*  **17-tree**: feat: create welcome/to/school directory strucutre.
